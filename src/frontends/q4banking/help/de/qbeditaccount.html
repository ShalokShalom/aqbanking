Kontoeinstellungen ändern

Dieser Dialog ermöglicht die Konfiguration eines Kontos.

Dieser Dialog enthält verschiedene Seiten:

  • Einleitung (enthält allgemeine Hinweise zum Dialog) [immer vorhanden]
  • Grundeinstellungen (allgemeine Einstellungen zum Konto) [immer vorhanden]
  • eine optionale Anwendungs-Seite (z.B. QBankManager, hier können Sie - falls diese Seite vorhanden ist - Anwendungs-spezifische Einstellungen vornehmen)
  • eine optionale Seite des Banking-Moduls, welches für das bearbeitete Konto zuständig ist, z.B.:
    • HBCI (HBCI-spezifische Einstellungen)
    • DTAUS (Einstellungen zu DTAUS)

Bank-Einstellungen

Hier werden Einstellungen bezüglich der Bank vorgenommen.
Feld Erläuterung
Staat Wählen Sie den Staat, in dem die Bank ihren Sitz hat.
Bankleitzahl Geben Sie hier für deutsche Banken die Bankleitzahl ein. Für Staaten, in denen keine Bankleitzahlen üblich sind (bzw. falls Sie sie nicht kennen), geben Sie hier 00000000 ein.

Verfügbare Benutzer / ausgewählte Benutzer

Manche Banking-Module arbeiten Benutzer-orientiert (wie HBCI, OFX etc). Diese Module müssen wissen, unter welchem Benutzer Sie sich bei der Bank anmelden wollen, wenn Sie Aufträge ausführen wollen.

Auf der linken Seite (verfügbare Benutzer) finden Sie die Benutzer, die Sie dem Konto zuordnen können.

Auf der rechten Seite finden Sie die Benutzer, die bereits dem Konto zugeordnet wurden (entweder durch Sie oder automatisch durch das Banking-Modul).

Konto-Einstellungen für HBCI

Hier können Sie HBCI-spezifische Einstellungen an Ihrem Konto vornehmen.

Allgemeine Einstellungen

Feld Erläuterung
Einzelüberweisung anstelle Sammelüberweisung bevorzugen

Wenn dieses Feld markiert ist, werden bei Überweisungen immer Einzelaufträge erzeugt, andernfalls Sammelaufträge.

Sammelaufträge sind bei manchen Banken preiswerter als Einzelaufträge. Allerdings haben Sammelaufträge den Nachteil, daß man je nach Bank oft nur einen Gesamtposten in seinen Kontoauszügen findet, wodurch die Zuordnung der Umsätze zu den einzelnen Überweisungen unmöglich wird.

Einzellastschrift anstelle Sammellastschrift bevorzugen Wie das obige Feld, allerdings für Lastschriften.

Konto-Einstellungen für DTAUS

Hier können Sie DTAUS-spezifische Einstellungen an Ihrem Konto vornehmen.

DTAUS Einstellungen

Feld Erläuterung
Maximale Anzahl Verwendungszweckzeilen Zulässig sind meistens 2-6.
Lastschriften erlauben Markieren Sie dieses Feld, falls Ihre Bank Ihnen das Einziehen von Lastschriften erlaubt und Sie dies tun möchten.
Alle Umsätze ausdrucken Falls dieses Feld markiert ist, werden auf dem Begleitzettel zur DTAUS-Diskette alle Überweisungen ausgedruckt. Andernfalls wird nur eine Übersicht ausgedruckt.

Ablage-Einstellungen

Feld Erläuterung
Zielverzeichnis

Geben Sie hier das Verzeichnis an, in das Sie die DTAUS-Dateien speichern wollen.

Falls Sie direkt DTAUS-Disketten erstellen wollen - statt die Dateien auf der Festplatte abzulegen - sollten Sie hier den Pfad angeben, unter dem das Diskettenlaufwerk zu finden ist. Meist ist dies /floppy, /mnt/floppy oder /media/floppy (je nach System).

Klicken Sie auf den Knopf neben dem Eingabefeld um einen Datei-Dialog zu erhalten.

Diskette verwenden Markieren Sie dieses Feld, falls Sie direkt Disketten beschreiben wollen. In diesem Fall stellt das Modul sicher, daß sich in dem angegebenen Verzeichnis am Ende nur eine einzige Datei befindet, die außerdem den von DTAUS vorgeschriebenen Namen trägt.
Einbinden erlauben

Das Modul kann automatisch Disketten im Laufwerk einbinden.

Die eingelegt Diskette muß im MS-DOS-Format formatiert und leer sein.

Falls dieses Feld markiert ist, müssen Sie die Kommandos zum Einbinden in den nächsten Feldern eingeben.

Bitte beachten Sie, daß dieses Feld ausgeschaltet sein muß, falls Ihr System einen automounter verwendet.

Kommando zum Einbinden

Hier geben Sie nun das Kommando zum Einbinden einer Diskette im Laufwerkes an. Die nötige Eingabe hängt von Ihrem System ab.

Für Linux müssen Sie hier das folgende Kommando eingeben:

mount Zielverzeichnis wie oben angegeben

Falls Sie also oben /floppy eingegeben haben, lautet das korrekte Kommando:

mount /floppy
Lösen der Einbindung

Geben Sie hier analog zum vorigen Feld das Kommando zum Lösen der Einbindung der Diskette ein.

Für Linux müssen Sie hier das folgende Kommando eingeben:

umount Zielverzeichnis wie oben angegeben

Falls Sie also oben /floppy eingegeben haben, lautet das korrekte Kommando:

umount /floppy
